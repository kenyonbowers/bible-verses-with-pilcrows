# bible-verses-with-pilcrows
This is a json and a txt file containing every verse in the King James Bible that contains a pilcrow.

## json format
```json
{
  "Genesis": ["1:6", "1:9", ...],
  "Exodus": ["1:7", "1:15", ...],
  ...
}
```

## txt format
```
[Genesis]
1:6, 1:9, ...

[Exodus]
1:7, 1:15, ...

...
```
