# Bible Verses Containing Pilcrows
These files (.JSON and .TXT) contain a list of every verse in the King James Bible which contain a pilcrow.

## .JSON Format
```json
{
  "Genesis": ["1:6", "1:9", ...],
  "Exodus": ["1:7", "1:15", ...],
  ...
}
```

## .TXT Format
```
[Genesis]
1:6, 1:9, ...

[Exodus]
1:7, 1:15, ...

...
```
